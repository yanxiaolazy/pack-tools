## pack-tools
simple packing tool

### Installation
```bash
$ yarn install
```
### Production
```bash
$ yarn run build
```
### Development
```bash
$ yarn run start
```
### Description
To use `windows`, please modify `package.json` 
```json
"script": {
  "start": "set NODE_ENV=development && webpack ...",
  "build": "set NODE_ENV=production && webpack ..."
}
```
