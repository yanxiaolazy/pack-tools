## pack-tools
simple packing tool

### Installation
```bash
$ yarn install
```
### Production
```bash
$ yarn run build
```
### Development
```bash
$ yarn run start
```
### Contribute
* Fork this Repo first
* Clone your Repo
* Install dependencies by $ yarn install
* Checkout a feature branch
* Feel free to add your features
* Make sure your features are fully tested
* Publish your local branch, Open a pull request
* Enjoy hacking <3
